Main Path Mapping to FSM States

Path Condition Main States Output Intent
Rx pattern recogniser rxnow='1' State-independent parser update in rx_parser_regs Detect ANNN/aNNN, L/l, P/p and queue event flags
Data echoing echo_pending='1' in IDLE ECHO_TX → ECHO_TX_WAIT Transmit received byte back over UART
L printing list_evt='1' and result_valid='1' LIST_NL_LF → LIST_NL_CR → byte-loop states Transmit full result list from snapshot registers
P printing peak_evt='1' and result_valid='1' PEAK_NL_LF → PEAK_NL_CR → peak/index states Transmit peak byte plus 3-digit BCD index
