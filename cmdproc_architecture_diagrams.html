Command Processor Architecture Diagrams

Synced to the current Cmdtb/cmdProc.vhd implementation: exact FSM state names, RX echo priority, and snapshot latching of dataResults/maxIndex on seqDone.

Block-Level Sketch (Implemented)

Four-process implementation: state_reg, rx_parser_regs, next_state_logic, and output_logic. 

flowchart LR
  RX["UART Rx\nrxnow, rxData"] --> RXP["rx_parser_regs\n4-byte window (rxWin0..rxWin3)\ncommand detect + echo capture"]
  RXP --> EVT["Event/state regs\nstart_evt/list_evt/peak_evt\necho_pending + numWords_reg"]

  DP["Data processor\nbyte, dataReady, seqDone,\ndataResults, maxIndex"] --> RXP
  RXP --> SNAP["Result snapshot regs\nresult_bytes_reg\npeak_index_reg\nresult_valid + seqDone_latched"]

  ST["state_reg\ncurState"] --> NS["next_state_logic\n(transition arbitration)"]
  EVT --> NS
  SNAP --> NS
  NS --> ST

  ST --> OUT["output_logic\nstart, txData, txnow, rxdone"]
  EVT --> OUT
  SNAP --> OUT
  OUT --> TX["UART Tx handshake\npulse txnow, wait txdone"]

FSM State Diagram (Exact Names)

IDLE priority: echo_pending > start_evt > list_evt > peak_evt.

Transition style: next_state_logic defaults to hold state; wait states only move on txdone=1. 

stateDiagram-v2
  [*] --> IDLE

  IDLE --> ECHO_TX: echo_pending=1
  ECHO_TX --> ECHO_TX_WAIT
  ECHO_TX_WAIT --> IDLE: txdone=1

  IDLE --> RUN_NL_LF: start_evt=1
  RUN_NL_LF --> RUN_NL_LF_WAIT
  RUN_NL_LF_WAIT --> RUN_NL_CR: txdone=1
  RUN_NL_CR --> RUN_NL_CR_WAIT
  RUN_NL_CR_WAIT --> RUN_REQ: txdone=1

  RUN_REQ --> IDLE: seqDone_latched=1
  RUN_REQ --> RUN_TX_HI: dataReady=1
  RUN_REQ --> RUN_REQ: otherwise
  RUN_TX_HI --> RUN_TX_HI_WAIT
  RUN_TX_HI_WAIT --> RUN_TX_LO: txdone=1
  RUN_TX_LO --> RUN_TX_LO_WAIT
  RUN_TX_LO_WAIT --> RUN_TX_SEP: txdone=1
  RUN_TX_SEP --> RUN_TX_SEP_WAIT
  RUN_TX_SEP_WAIT --> RUN_REQ: txdone=1

  IDLE --> LIST_NL_LF: list_evt=1 and result_valid=1
  LIST_NL_LF --> LIST_NL_LF_WAIT
  LIST_NL_LF_WAIT --> LIST_NL_CR: txdone=1
  LIST_NL_CR --> LIST_NL_CR_WAIT
  LIST_NL_CR_WAIT --> LIST_TX_HI: txdone=1
  LIST_TX_HI --> LIST_TX_HI_WAIT
  LIST_TX_HI_WAIT --> LIST_TX_LO: txdone=1
  LIST_TX_LO --> LIST_TX_LO_WAIT
  LIST_TX_LO_WAIT --> LIST_TX_SEP: txdone=1
  LIST_TX_SEP --> LIST_TX_SEP_WAIT
  LIST_TX_SEP_WAIT --> LIST_TX_HI: txdone=1 and list_idx < RESULT_BYTE_NUM-1
  LIST_TX_SEP_WAIT --> IDLE: txdone=1 and list_idx = RESULT_BYTE_NUM-1

  IDLE --> PEAK_NL_LF: peak_evt=1 and result_valid=1
  PEAK_NL_LF --> PEAK_NL_LF_WAIT
  PEAK_NL_LF_WAIT --> PEAK_NL_CR: txdone=1
  PEAK_NL_CR --> PEAK_NL_CR_WAIT
  PEAK_NL_CR_WAIT --> PEAK_TX_HI: txdone=1
  PEAK_TX_HI --> PEAK_TX_HI_WAIT
  PEAK_TX_HI_WAIT --> PEAK_TX_LO: txdone=1
  PEAK_TX_LO --> PEAK_TX_LO_WAIT
  PEAK_TX_LO_WAIT --> PEAK_TX_SPACE: txdone=1
  PEAK_TX_SPACE --> PEAK_TX_SPACE_WAIT
  PEAK_TX_SPACE_WAIT --> PEAK_TX_I2: txdone=1
  PEAK_TX_I2 --> PEAK_TX_I2_WAIT
  PEAK_TX_I2_WAIT --> PEAK_TX_I1: txdone=1
  PEAK_TX_I1 --> PEAK_TX_I1_WAIT
  PEAK_TX_I1_WAIT --> PEAK_TX_I0: txdone=1
  PEAK_TX_I0 --> PEAK_TX_I0_WAIT
  PEAK_TX_I0_WAIT --> IDLE: txdone=1
